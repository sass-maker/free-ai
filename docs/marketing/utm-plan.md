# Free AI UTM Plan

## Naming Rules

- Use lowercase words with hyphens.
- Keep `utm_source` to the platform or referrer.
- Keep `utm_medium` to the channel type.
- Keep `utm_campaign` to the marketing bet.
- Use `utm_content` for hook, format, or asset variant.
- Use `utm_term` only for paid search or keyword tests.

## Source And Medium

| Source | Medium | Use |
| --- | --- | --- |
| tiktok | short-video | AI-generated demo reels |

| github | repo | README and issue comments |
| docs | owned | Internal docs links |
| newsletter | email | Founder and update emails |

## Campaigns

| Campaign | Goal |
| --- | --- |
| free-first-gateway | Explain free-first OpenAI-compatible routing |
| prototype-without-bill | Reach builders worried about early model spend |
| openai-compatible-switch | Capture users already using OpenAI SDKs |

| first-request | Push visitors to make one successful request |

## Dashboard Questions

- Which campaign creates the most API key requests?
- Which content variant leads to first successful request fastest?
- Which source brings users who return after 24 hours?
- Which medium creates the most docs-to-playground movement?
- Which campaign attracts high-error traffic that needs better setup copy?
